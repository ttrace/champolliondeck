# Pre-Babel Lens for macOS 0.8.0 リリースノート

## 主な新機能

- Clutch連動（Source / Output）を追加
  - Source側で選択したセグメントに対応するOutput側セグメントをハイライト
  - Output側の選択からSource側への逆引きハイライトにも対応
  - 対応箇所が画面外の場合は自動スクロールで表示

- 表示メニューを強化
  - コンパクト表示 / カラム表示の切り替えを追加
  - 標準サイズ / 拡大 / 縮小（文字サイズ変更）を追加

- メニューの多言語対応を拡充
  - 日本語 / 英語 / 韓国語 / 中国語（簡体字 / 繁体字）に対応

- Vision Framework を利用した画像OCRに対応
  - 画像からテキストを抽出して Source へ取り込み可能
  - OCR結果をそのまま翻訳フローに連携

## 改善

- Clutch操作時のハイライト追従とスクロール挙動を改善し、対照読解の操作感を向上
- 表示切り替え時の安定性を改善

## 補足

- 翻訳処理は引き続きオンデバイスで実行されます
- 一部言語では初回利用時に、システム管理の言語データ取得が必要な場合があります
