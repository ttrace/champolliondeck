# Pre-Babel Lens v0.8.0 (Draft)

## User-facing update information / ユーザー向けアップデート情報

This release focuses on editing-and-reading flow quality across iOS and macOS: better segment-to-segment clutch navigation, more stable compact-reader behavior, stronger fallback resilience, and improved iOS share import support for document files.
このリリースでは、iOS/macOS の編集・閲覧フローを中心に改善しました。セグメント連動（Clutch）の操作性、コンパクト読書モードの安定性、フォールバック時の完走性、iOS共有取り込み（文書ファイル）の対応を強化しています。

## Highlights

- Clutch mode: bidirectional segment highlighting on iOS and macOS
  - Tapping/cursoring a segment in Source highlights the corresponding Output segment.
  - Selecting an Output segment also highlights the corresponding Source range.
  - Out-of-view targets are auto-scrolled into view with reduced jitter.
  - iOS/macOS の両方で、Source/Output の相互セグメントハイライトに対応しました。
  - 画面外ターゲットは自動スクロールし、従来よりジッターを抑制しています。

- Compact stacked layout improvements (iOS-first behavior)
  - Reader-mode transitions were refined to better preserve intent during scroll.
  - Source field stability and output readability were improved during compact interactions.
  - コンパクト縦積み時の読書モード遷移を見直し、スクロール時の意図しない挙動を減らしました。
  - コンパクト操作中の Source 安定性と Output 可読性を改善しました。

- Translation fallback and retry robustness
  - Added timeout/retry guardrails for Translation Framework recovery path.
  - Added stale pending request recovery to reduce stuck translation sessions.
  - Translation Framework の回復経路に timeout/retry ガードを追加しました。
  - pending リクエストの滞留回復を追加し、翻訳停止状態に陥りにくくしました。

- iOS Share Panel: better document import
  - Added share ingestion support for `.pdf` and `.docx` content extraction.
  - PDF extraction applies soft line-break normalization for easier reading/editing.
  - iOS Share Panel から `.pdf` / `.docx` のテキスト抽出取り込みに対応しました。
  - PDF 取り込み時はソフト改行を正規化し、読みやすさを改善しています。
