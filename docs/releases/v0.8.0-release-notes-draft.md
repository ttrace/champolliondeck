# Pre-Babel Lens v0.8.0 (Draft)

## User-facing update information / ユーザー向けアップデート情報

This release adds new interaction features across macOS and iOS: new View-menu controls on macOS (compact mode, column mode, and text-size actions), Clutch segment linking between Source and Output, Vision Framework-based image OCR, and document import support from the iOS Share Panel.
このリリースでは、macOS/iOS に新しい操作機能を追加しました。macOS の表示メニュー拡張（コンパクト/カラム表示、文字サイズ変更）、Source/Output 連動の Clutch、Vision Framework を利用した画像OCR、iOS Share Panel の文書取り込みに対応しています。

## Highlights

- New feature: Clutch mode on iOS and macOS
  - Tapping/cursoring a segment in Source highlights the corresponding Output segment.
  - Selecting an Output segment also highlights the corresponding Source range.
  - Out-of-view targets are auto-scrolled into view.
  - iOS/macOS の両方で、Source/Output の相互セグメントハイライトに対応しました。
  - 画面外ターゲットの自動スクロールに対応しています。

- New feature: macOS View-menu controls
  - Added compact mode and column mode switching.
  - Added text-size actions: Actual Size / Zoom In / Zoom Out.
  - Added localized menu labels for Japanese, English, Korean, Simplified Chinese, and Traditional Chinese.
  - macOS の表示メニューに、コンパクト/カラム表示の切り替えを追加しました。
  - 文字サイズ変更（標準サイズ/拡大/縮小）を追加しました。
  - メニュー文言を日本語・英語・韓国語・中国語（簡体字/繁体字）に対応しました。

- New feature: iOS Share Panel document import
  - Added share ingestion support for `.pdf` and `.docx` text extraction.
  - PDF extraction applies soft line-break normalization.
  - iOS Share Panel から `.pdf` / `.docx` のテキスト抽出取り込みに対応しました。
  - PDF 取り込み時にソフト改行正規化を適用します。

- New feature: Vision Framework image OCR
  - Added image-to-text OCR using Vision Framework.
  - OCR text can be imported into Source and translated immediately.
  - Vision Framework を利用した画像OCR（画像からのテキスト抽出）に対応しました。
  - 抽出したテキストは Source に取り込み、続けて翻訳できます。
