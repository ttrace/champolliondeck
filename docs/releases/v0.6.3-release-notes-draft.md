# Pre-Babel Lens v0.6.3 (Draft)

## User-facing update / ユーザー向けアップデート

This release improves unsafe fallback behavior on iPhone and keeps translation output clearer when fallback is triggered.
このリリースでは、iPhone の unsafe フォールバック挙動を改善し、フォールバック時の出力可読性を高めました。

## Highlights

- iOS Translation Framework fallback
  - When Apple Intelligence safety guardrails block a segment, the app now attempts an on-device Translation Framework fallback first.
  - Apple Intelligence の安全判定で翻訳不能になったセグメントに対して、まず on-device の Translation Framework 補完を試みます。
- Safer fallback behavior remains intact
  - If Translation Framework cannot translate the segment, the app keeps the existing source-text fallback behavior.
  - Translation Framework でも翻訳できない場合は、従来どおり原文フォールバックを維持します。
- Clearer unsafe marker colors
  - Segments recovered by Translation Framework are highlighted in light blue.
  - Source-text fallback segments keep the existing warning color.
  - Translation Framework で補完できたセグメントは水色で表示されます。
  - 原文フォールバックは従来の警告色のままです。

## Notes

- Translation processing is on-device.
- Some language resources may require an initial system-managed download.
- 翻訳処理は on-device で行われます。
- 一部言語では初回にシステム管理の言語データダウンロードが必要です。
