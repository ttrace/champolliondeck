# zen-Babel for macOS 0.8.1 リリースノート

## 主な変更

- アプリ名を変更
  - `Pre-Babel Lens` から `zen-Babel` に改名しました。

- Source への画像ペーストOCRに対応
  - Pasteボタンで画像を貼り付けた場合にOCRを実行します。
  - Sourceフィールドへの通常ペースト（`⌘V` / コンテキストメニュー）でも画像OCRを実行します。
  - 抽出したテキストをそのまま翻訳フローに連携できます。

## 改善

- Viewメニューのコンパクト/カラム切り替え文言を調整
  - Issue: #1
  - 表示モード切り替え時のラベル整合性を改善しました。

## 補足

- 翻訳処理は引き続きオンデバイスで実行されます。
- Bundle ID / URLスキーム（`prebabellens://`）は互換性のため維持しています。
