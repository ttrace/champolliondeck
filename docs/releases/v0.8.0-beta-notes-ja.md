# Pre-Babel Lens v0.8.0 ベータ版のご案内

Pre-Babel Lens v0.8.0 では、macOS / iOS の操作性を中心に新機能を追加しました。

## 主な新機能

1. Clutch 連動（iOS / macOS）
- 原文と訳文を対照表示する機能、クラッチを搭載しました。
- Source 側でタップ・カーソル移動したセグメントに対応する Output セグメントをハイライトします。
- Output 側で選択したセグメントに対応する Source 範囲もハイライトします。
- 対応箇所が画面外にある場合は、自動でスクロールして表示します。

2. 文字サイズ変更
- Source、Outputのいずれかでピンチイン・アウトをすると文字のサイズを5段階に調整できます
- また表示メニュー文言の多言語化（日本語 / 英語 / 韓国語 / 中国語 簡体字 / 中国語 繁体字）に対応しました。

3. iOS Share Panel からの文書取り込み
- Share Panel 経由で `.pdf` / `.docx` のテキスト取り込みに対応しました。
- PDF 取り込み時は、読みやすさを優先してソフト改行の正規化を行います。

## テストで確認していただきたい点

- Clutch 連動時のハイライトと自動スクロールの自然さ
- macOS の表示メニュー操作（コンパクト / カラム表示、文字サイズ変更）
- iOS Share Panel からの PDF / DOCX 取り込み結果
- 翻訳不能言語や言語パック未導入時のメニュー表示とガイド表示

## 補足

- 翻訳処理は引き続きオンデバイスで実行されます。
- 一部言語は初回利用時に、システム管理の言語データダウンロードが必要な場合があります。
