# Pre-Babel Lens v0.4.0 (Draft)

## Performance update

More stable, quick response and better quality, developer hope so.

## Highlights inside

- Added segment `kind` inference in preprocessing:
  - `heading`
  - `dialogue`
  - `ui-labels`
  - `lists`
  - `general`
- Improved Foundation Models structured generation with typed schema:
  - adopted `@Generable` / `@Guide`
  - reduced prompt dependency for output structure
- Improved translation observability:
  - per-segment logs for `preprocess-kind` and `structured-kind`
  - session-scoped logs (`[Sx]`) with timing metrics
- Improved Developer Console usability:
  - copy-friendly text field
  - `Clear` button
- Improved resiliency for malformed/placeholder structured output:
  - retry path tightened
  - segment-level fallback to source text when needed

## Notes

- This release targets Apple Intelligence-enabled environments.
- Foundation Models availability and model asset readiness are still required at runtime.

## Verification

- Automated tests: `swift test` (passed)
- Manual verification (recommended before final publish):
  - short and long translation inputs
  - multiple target languages
  - per-segment `kind` log output
  - fallback behavior under malformed structured responses
